# 자료형과 기본 연산


```python
a = 1 # 정수형
b = 1 # 정수형
print(a, b)
```

    1 1
    


```python
a = 1.5
b = 3.5
print(a, b)
```

    1.5 3.5
    


```python
a = "ABC"    # 문자형
b = "101010" # 문자형
print(a, b)
```

    ABC 101010
    


```python

b = False
print(a, b)
```

    True False
